# 通用表达模板 / General Expression Templates

Use these only when no user-provided template or audience/object guide fits.

## Explanatory

Use when the goal is understanding a concept, mechanism, or judgment.

Structure: what it is -> why it matters -> how it works -> common misunderstandings -> where it applies and where it does not -> short takeaway.

## Persuasive

Use when the goal is alignment, buy-in, or pushing a recommendation.

Structure: current problem -> cost or risk of not acting -> available paths -> recommended path -> why now -> next step.

## Training

Use when the goal is helping beginners learn.

Structure: learning objective -> background context -> step-by-step explanation or operation -> example -> common mistakes -> checklist or practice questions.

## Operational

Use when the goal is execution or SOP.

Structure: applicable scenario -> prerequisites -> steps -> decision branches -> exception handling -> acceptance criteria.

## Decision

Use when the audience needs to decide.

Structure: core conclusion -> options compared -> cost and benefit -> risk -> recommended decision -> open questions for approval.

## Retrospective

Use when summarizing a project, experiment, or process.

Structure: background -> what was done -> result -> problems -> lessons -> next step.
